# Healthcare Connect

Healthcare Connect is a modern and responsive web application designed for an NGO to bridge the gap between patients in need and volunteers/medical resources.

## 🚀 Features
- **Modern Landing Page**: Professional healthcare-themed UI with smooth animations.
- **Patient Support Form**: Comprehensive request form with client-side validation.
- **Local Storage Integration**: Persistent data storage in the browser to track support requests.
- **Impact Dashboard**: Real-time counter of total requests submitted.
- **AI FAQ Chatbot**: Keyword-based virtual assistant for quick queries.
- **Dark/Light Mode**: User-preferred theme persistence.
- **Fully Responsive**: Optimized for mobile and desktop screens.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with Flexbox/Grid, transitions, and dark mode variables.
- **JavaScript**: ES6+ logic for form handling, local storage, and chatbot behavior.
- **Font Awesome**: iconography.
- **Google Fonts**: Inter typography.

## 🤖 AI/Automation Idea
The **Health Bot** is a keyword-matched AI interface that provides immediate answers to common logistical questions. 
- **Use Case**: Reduces the workload on NGO staff by handling basic queries about working hours, contact info, and required documentation.
- **Disclaimer**: Always reminds users that it is for informational purposes only and does not provide medical diagnosis.

## 🏢 NGO Use Case
This application serves as a digital touchpoint for a small to medium-sized NGO. It allows them to:
1. Digitize patient request collection.
2. Track their community impact through a visual dashboard.
3. Provide 24/7 automated basic support through the chatbot.
